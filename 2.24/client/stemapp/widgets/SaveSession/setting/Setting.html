Use Server To Download File
Check this box to have the widget send the session data to the server when saving a file. All version of IE require using the server to download a file. If checked, you must enter a Url to Save the File.
Url for Save To File
The url that will return the file to download
Filename for All Sessions
The file name to use when the Save to File button is clicked
Filename for 1 Session
The file name to use when the download action in the Save Session table is clicked. This can be a template that includes placeholders for properties from the session such as "{name}"
