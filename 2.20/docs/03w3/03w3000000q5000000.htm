What's new in version 2.18

The ArcGIS Web AppBuilder (Developer Edition) 2.18 update includes the following widget enhancements.

Widgets

  • Business Analyst—You can now customize the content for the widget's home screen. You can also configure export options for infographics.
  • Threat Analysis—The threat type categories now include liquid propane gas (LPG) threats.
