What's new in version 2.17

The ArcGIS Web AppBuilder (Developer Edition) 2.17 update includes the following widget enhancements.

Widgets

  • Business Analyst—You can now specify a default infographic and report when configuring the widget.
  • Parcel Drafter—You can now configure and edit additional attributes.
  • Smart Editor—There's a new configuration option to honor web map settings if you want to always synchronize the related settings defined in the web map. You can also configure the widget to automatically expand related records for quick editing. When specifying attribute actions presets, you can now select valid date values from the data. Intersection Attribute Actions provide an option to enable a value picker for end users to select between multiple intersecting values.
