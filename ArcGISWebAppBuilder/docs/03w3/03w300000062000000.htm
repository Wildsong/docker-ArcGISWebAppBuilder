What's new

The ArcGIS Web AppBuilder (Developer Edition) 2.19 update includes the following features and enhancements.

General

  • Support for Slovak language.

Widgets

  • Add Data—You can now add WMTS OGC web services.
  • Gridded Reference Graphic—UTM has been added to the reference system options.
  • Parcel Drafter—For unclosed parcels created by the widget in this release and later, users can now edit and close the unclosed boundaries.
  • Smart Editor—You can now use Military Grid Reference System (MGRS) when moving point features to a new location using coordinates.
